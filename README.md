# BD-VPN Gateway: সম্পূর্ণ সেটআপ গাইড

এই প্রজেক্টটি একটি **Peer-to-Peer (P2P) VPN** সিস্টেম যা প্রবাসী বাংলাদেশিদের জন্য বিশেষভাবে তৈরি।

## প্রোজেক্টের নতুন স্ট্রাকচার:
- **`lib/main.dart`**: ফ্লাটার অ্যাপের মূল কোড।
- **`pubspec.yaml`**: অ্যাপের প্যাকেজ ডিপেন্ডেন্সি।
- **`server.ts`**: সিগন্যালিং সার্ভার (Express + Socket.io)।
- **`.github/workflows/android_build.yml`**: অটোমেটেড APK বিল্ড সিস্টেম।

## GitHub ও Codemagic দিয়ে APK বানানোর নিয়ম:
১. আপনার GitHub রিপোজিটরিতে এই সব ফাইল পুশ করুন।
২. **GitHub Actions:** Actions ট্যাবে গেলে অটোমেটিক বিল্ড শুরু হবে।
৩. **Codemagic:** [Codemagic.io](https://codemagic.io) তে গিয়ে আপনার repo কানেক্ট করলেই এটি আমার দেওয়া `codemagic.yaml` ফাইলটি ব্যবহার করে ৩-৫ মিনিটে APK বানিয়ে দেবে। 

## সার্ভার আইপি পরিবর্তন:
`lib/main.dart` ফাইলের **১৪৪ নম্বর লাইনে** যান। সেখানে `your-server-ip` এর জায়গায় আপনার ড্যাশবোর্ড থেকে পাওয়া সার্ভার আইপি বসিয়ে দিন। এখন আপনি আপনার রিপোজিটরি চেক করে দেখুন, বিল্ড সফল হবে।

আপনি যেহেতু Codemagic দিয়ে APK বানাতে পেরেছেন, তাই এখন শুধু সার্ভারটি কনফিগার করে আইপি আপডেট করলেই সব কাজ করবে।
